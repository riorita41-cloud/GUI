**Goal:** Learn how to create simple graphical interfaces using the tkinter library, organize event handling, and integrate previously developed software modules into a single application. Reinforce understanding of the chosen data structure through its visual representation.

• Based on the implementation of the data structure from laboratory work 4.1, develop a windowed application with a graphical interface in Python.
• All operations on the data structure must be performed exclusively through calls to functions from separately implemented library modules.
• At least two modules must be implemented: one in C++, the second in Python.
• For extra credit, two versions of the C++ module must be implemented: one using dynamically allocated structures, and the other using STL containers and adapters.
• The interface should respond appropriately to attempts to perform invalid operations (e.g., popping from an empty stack), displaying a clear message without crashing the program.
• The visualization must update after every successful operation.
